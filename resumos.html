Branches e Hospedagem

Branches

São ramificações de um projeto de software, tendo por objetivo o desenvolvimento paralelo de:

  1. Novas funcionalidades
  2. Correções de erros
  3. Alteração de código

Tipos de branches:

  • Branch Master (Main): Braço principal de um projeto em um repositório.
  • Branch Secundária: Braço secundário de um projeto em um repositório.

Passo a passo:

  • No GitHub Desktop crie um novo repositório.
  • Publique o seu repositório (Ctrl+P).
  • Abra seu repositório no Visual Studio e crie um arquivo index.html.
  • Edite o arquivo index.html como desejar.
  • Realize o commit to main.
  • Até aqui você editou a branch master.
  • Acesse Current Branch e crie um nova branch.
  • New Branch --> Edite o nome como secundaria-01 --> Create branch.
  • Publique sua branch Secundária
  • Se desejar você pode editar a branch secundária.
  • Crie uma segunda branch secundária.
  • Faça as alterações que desejar em sua segunda branch secundária.

Realizando o Merge:

Realizar um Merge é fazer a junção de um Branch secundário em um Branch Master.

  • Selecione a Branch que deseja realizar o Merge --> Menu --> Branch --> Merge into current branch (Ctrl+Shift+M).
  • Selecione quem você deseja realizar o Merge na Branch selecionada.
  • OBS: Se houver algum conflito, clique em Open on Visual Studio e selecione uma opção (Manter as duas alterações, manter a primeira alteração, manter a segunda alteração).
  • Para continuar usando a Branch secundária após o Merge: Menu --> Branch --> Update to main (Ctrl+Shift+U)

Criando uma Branch "Terciaria":

Para isso, basta criar uma Branch apartir de sua Branch Secundária.

Hospedagem

Serve para consolidar o trabalho realizado.

O GitHub possui a funcionalidade do GitHub Pages, que realiza a hospedagem gratuita de repositórios públicos.

Sua URL será: https:/nomeusuario.github.io/repositório

Passo a Passo:

  • Abra seu repositório no GitHub.
  • Settings --> Pages.
  • Source --> None --> Selecione o main --> Save.
  • PRONTO! Basta atualizar a página (F5) e acessar seu site.

Agora que você publicou o seu site, deve tomar alguns cuidados como não editar o Branch Main diretamente, sempre editar um Branch Secundario, testar e depois realizar o Merge.

Tags Especiais

Tag <table>

Usada para criar tabelas em um site html.

<tr>: Define uma linha.

<th>: Define o cabeçalho.

<td>: Define o conteúdo de uma célula da tabela.

Exemplo de tabela:

Nome Sobrenome
João Santos
Maria Aparecida

Tag <del>

Usada para mostrar elementos "deletados".

OBS: Estes elementos apareceram riscados no navegador.

Exemplo de elemento deletado:

O limão é laranja.

Tag <ins>

Usada para mostrar elementos "inseridos".

OBS: Estes elementos serão mostrados sublinhados.

Exemplo de elemento inserido:

O limão é laranja verde

Tag <sub>

Para escrever palavras subscritas, ou seja, meia linha abaixo do texto normal, pode aparecer em uma fonte menor.

Exemplos de elementos subscritos:

H2O

CH4

Tag <sup>

Para escrever palavras sobrescritas, ou seja, meia linha acima, também pode apresentar uma fonte menor.

Exemplos de elementos sobrescritos:

32 = 9

23 = 8

Tag <blockquote>

Sua tradução literal é 'bloco de citação'.

É usado para realizar alguma citação em um site.

Normalmente colocado em um recuo do texto principal.

Exemplo de bloco de citação:

Ser ou não ser, eis a questão, será mais nobre Em nosso espírito sofrer pedras e setas Com que a Fortuna, enfurecida, nos alveja, Ou insurgir-nos contra um mar de provocações E em luta pôr-lhes fim? Morrer… dormir: não mais. - William Shakespeare.

Tag <q>

Espécie de bloco de citação curto.

Ao invés de sofrer um recuo, a frase aparece entre aspas.

Exemplo de citação curta:

William Shakespeare: Ser ou não ser, eis a questão.

Tag <abbr>

Para escrever uma abreviação e mostrar o seu nome completo.

Além da tag propriamente, o elemento title pode ser utilização para descrever o 'título' completo da abreviação.

A abreviação aparece pontilhada, como no exemplo.

Exemplo de como utilizar uma abreviação:

IFPR - Campus Irati.

Tag <address>

Usada para mostrar informações do contato de uma ator ou proprietário.

Como endereço, telefone, email ou rede social.

Mostrado em itálico e possui uma quebra de linha antes e depois.

Exemplo de utilização:

Escrito por: Carlos Santos.
Email: carlos@email.
Endereço: Rua Ana da Luz, 122.

Tag <cite>

Usado para citar uma obra, livro, artigo ou poema, entre outros.

Formatado em itálico.

OBS: O nome do ator não é uma citação.

Exemplo de citação:

Harry Potter e a Pedra Filosofal por J. K. Rowling.

Tag <bdo>

Significa substituição bidirecional.

Ou seja é usado para "inverter" o texto.

Exemplo de utilização:

Olá mundo.
Olá mundo.

Tag <code>

Usado para utilizar um texto com a fonte de um código de computador.

Exemplo de código:

Antes:

if(){}

Depois:

if(){}

Tag <pre>

Realiza a pré-formatação.

Utiliza exatamente a mesma formatação do documento HTML, respeitando cada espaço e quebra de linha.

É exibido com uma fonte de largura fixa.

Exemplo de pré-formatação:

   Olá mundo
    Esse é um    texto   pré-formatado
Ideal para   textos grandes  e  poemas   .

Tag <kbd>

Em inglês kbd é a sigla para keyboard (teclado).

Usado para representar atalhos e "entradas" de teclado.

Exemplo de atalho:

Ctrl + C

Ctrl + V

Multimídias

Combinação entre diferentes forma de mídias, como: Texto, áudio, imagens, animações e vídeos na mesma apresentação.

Tag <iframe>

Usada para incorporar outro documento no seu documento HTML.

Sintaxe:

<iframe src="URL" height="altura" width="largura" title="Título"></iframe>

Aplicação:

Incorporar outro site dentro do site:


<iframe src="URL de um site" height="altura" width="largura" title="Título"></iframe>

Incorporar aúdio no site:

Para pesquisar música utilize o site: https://soundcloud.com/, vá em compartilhar e selecione incorporado.

properhustle · Beatles - Hey Jude

<iframe src="URL da música no SoundCloud" height="altura" width="largura" title="Título"></iframe>

Incorporar vídeos ao site:

OBS: Acesse o https://www.youtube.com/ vá em compartilhar na opção incorporar .


<iframe src="URL do vídeo do YouTube" height="altura" width="largura" title="Título"></iframe>

Tag <video>

Usada para inserir um vídeo local, presente em sua máquina.


<video poster="Caminho para imagem do vídeo" controls>
    <source src="Caminho para o vídeo" type="Tipo de vídeo">

Pode-se adicionar arquivos em diferentes formatos, caso a primeira opção não funcione.

Tag <audio>

Usada para inserir um aúdio armazenado localmente, pode-se utilizar a Biblioteca de Audios do Youtube, que possui músicas sem direitos autorais.


<audio preload="metadata" controls>
    <source src="Caminho para o audio" type="Tipo de audio">

Tag <picture>

Usada para redimensionar uma imagem de acordo com a largura da página, possibilirando acessar a página em diversos tamanhos de dispositivos, para isso são adicionados cópias da imagem em tamanho distintos.


<picture>
    <source media="(max-width:largura máxima em pixels)" srcset="Caminho para a imagem pequena">
    <source media="(max-width:largura máxima em pixels)" srcset="Caminho para a imagem média">
    <img src="Caminho para a imagem grande" alt="Legenda">
</picture>
