Tags Especiais

Tag <table>

Usada para criar tabelas em um site html.

<tr>: Define uma linha.

<th>: Define o cabeçalho.

<td>: Define o conteúdo de uma célula da tabela.

Exemplo de tabela:

Nome Sobrenome
João Santos
Maria Aparecida

Tag <del>

Usada para mostrar elementos "deletados".

OBS: Estes elementos apareceram riscados no navegador.

Exemplo de elemento deletado:

O limão é laranja.

Tag <ins>

Usada para mostrar elementos "inseridos".

OBS: Estes elementos serão mostrados sublinhados.

Exemplo de elemento inserido:

O limão é laranja verde

Tag <sub>

Para escrever palavras subscritas, ou seja, meia linha abaixo do texto normal, pode aparecer em uma fonte menor.

Exemplos de elementos subscritos:

H2O

CH4

Tag <sup>

Para escrever palavras sobrescritas, ou seja, meia linha acima, também pode apresentar uma fonte menor.

Exemplos de elementos sobrescritos:

32 = 9

23 = 8

Tag <blockquote>

Sua tradução literal é 'bloco de citação'.

É usado para realizar alguma citação em um site.

Normalmente colocado em um recuo do texto principal.

Exemplo de bloco de citação:

Ser ou não ser, eis a questão, será mais nobre Em nosso espírito sofrer pedras e setas Com que a Fortuna, enfurecida, nos alveja, Ou insurgir-nos contra um mar de provocações E em luta pôr-lhes fim? Morrer… dormir: não mais. - William Shakespeare.

Tag <q>

Espécie de bloco de citação curto.

Ao invés de sofrer um recuo, a frase aparece entre aspas.

Exemplo de citação curta:

William Shakespeare: Ser ou não ser, eis a questão.

Tag <abbr>

Para escrever uma abreviação e mostrar o seu nome completo.

Além da tag propriamente, o elemento title pode ser utilização para descrever o 'título' completo da abreviação.

A abreviação aparece pontilhada, como no exemplo.

Exemplo de como utilizar uma abreviação:

IFPR - Campus Irati.

Tag <address>

Usada para mostrar informações do contato de uma ator ou proprietário.

Como endereço, telefone, email ou rede social.

Mostrado em itálico e possui uma quebra de linha antes e depois.

Exemplo de utilização:

Escrito por: Carlos Santos.
Email: carlos@email.
Endereço: Rua Ana da Luz, 122.

Tag <cite>

Usado para citar uma obra, livro, artigo ou poema, entre outros.

Formatado em itálico.

OBS: O nome do ator não é uma citação.

Exemplo de citação:

Harry Potter e a Pedra Filosofal por J. K. Rowling.

Tag <bdo>

Significa substituição bidirecional.

Ou seja é usado para "inverter" o texto.

Exemplo de utilização:

Olá mundo.
Olá mundo.

Tag <code>

Usado para utilizar um texto com a fonte de um código de computador.

Exemplo de código:

Antes:

if(){}

Depois:

if(){}

Tag <pre>

Realiza a pré-formatação.

Utiliza exatamente a mesma formatação do documento HTML, respeitando cada espaço e quebra de linha.

É exibido com uma fonte de largura fixa.

Exemplo de pré-formatação:

   Olá mundo
    Esse é um    texto   pré-formatado
Ideal para   textos grandes  e  poemas   .

Tag <kbd>

Em inglês kbd é a sigla para keyboard (teclado).

Usado para representar atalhos e "entradas" de teclado.

Exemplo de atalho:

Ctrl + C

Ctrl + V
