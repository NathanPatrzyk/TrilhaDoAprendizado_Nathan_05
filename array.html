Array

Definição

Destinado a armazenar dados indexados sequencialmente.

Exemplos feitos no arquivo: array.js

Métodos de Arrays

  • VETOR.push(VALOR) : Adiciona um valor no final da lista.
  • VETOR.pop : Remove o ultimo elemento da lista.
  • VETOR.shift : Remove o primeiro elemento da lista.
  • VETOR.unshift(VALOR) : Adiciona um elemento no começo da lista.
  • VETOR.indexOf(VALOR) : Retorna a posição apartir de uma palavra.
