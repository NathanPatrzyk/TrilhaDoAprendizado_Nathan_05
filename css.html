Introdução a CSS

CSS- Cascading Style Sheet

O HTML apenas mostra o conteúdo, adicionando o CSS é possivel realizar a estilização de conteúdo de documentos. Definindo fontes, cores e espaçamento.

Sintaxe

seletor{
            propriedade: valor; /*comentário*/
            propriedade-subpropriedade: valor;
            }

Exemplo

p{
            color: blue;
            text-align: center;
                        }

Formas de Utilizar

Inline Incorporado Externo
São definidos dentro de cada tag HTML. São definidos no cabeçalho apenas dentro da tag head. São definidos em um arquivo externo.
Afetam somente a tag onde foi definida. Afetam somente o documento em que foi definido. Afetam todos os documentos linkados ao CSS.

Passo a Passo

  1. Abra seu projeto no VisualStudio.
  2. Crie uma branch secundária.
  3. Crie um arquivo .css.
  4. Acionar conteúdo e envolver com tags html.

Passo a Passo: Inline

  1. Selecione uma tag HTML.
  2. Dentro da tag escreva: <h1 style="color: green;">Texto</h1>.
  3. Outras tags: <h1 style="background-color: green;">Texto</h1>, <h1 style="font-size: 20px;">Texto</h1>

Passo a Passo: Incorporado

  1. Faça da mesma maneira, mas com a tag <style>Estilização</style>, que pode ser adicionada no head.
  2. Outras tags: <h1 style="font-family: (nome da fonte);">Texto</h1>

Passo a Passo: Externo

  1. Para usar o CSS externo, você precisa adicionar o seguinte código na tag head: <link href="(Caminho para um documento .css)" rel="stylesheet">
  2. Nesse documento você pode escrever todo o seu código CSS.
  3. Outras tags: font-size: (Tamanho da fonte em pixels);
