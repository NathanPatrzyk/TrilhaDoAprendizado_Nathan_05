Imagens e Listas

Imagens

Representação visual de pessoas, objetos e paisagens.

No ambito digital é a representação de uma imagem bidimensional que utiliza números binários codificados para permitir seu armazenamento e processamento, entre outros.

Formatos de Imagens

.JPG --> Photographic Experts Group.

.PNG --> Portable Networking Graphic.

Para icones: .SVG e .ICO

Marcação de Imagens

Tags de imagem e parâmetros que definem as imagens e propriedades

Sintaxe:

<img src="imagem.extensão" width="largura" height="altura" alt="descrição">

Apartir do HTML 5 --> Tags figure e figcaption

Sintaxe:

<figure>

    <img src="imagem.extensão" width="largura" height="altura" alt="descrição">
    <figcaption&Legenda</> 
<figure>

Marcação de Favicon

Listas

Marcação de Listas

As tags de lista são divididas em:

  1. Não-ordenadas
  2. Ordenadas
  3. Detalhadas

Listas Não-Ordenadas

Sintaxe

<ul>

    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
<ul/>

Compilação

  • Item 1
  • Item 2
  • Item 3

Listas Ordenadas

Sintaxe

<ol type="elemento">

    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
<ol/>

Compilação

  1. Item 1
  2. Item 2
  3. Item 3

Listas Detalhadas

Sintaxe

<dl>

    <dt>Item 1</li>
    <dd>Esse é o primeiro item</dd>
    <li>Item 2</li>
    <dd>Esse é o segundo item</dd>
<dl/>

Compilação

Item 1
Esse é o primeiro item
Item 2
Esse é o segundo item
