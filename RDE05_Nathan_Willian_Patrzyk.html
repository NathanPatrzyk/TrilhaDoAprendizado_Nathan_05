Página Web e Website

Página Web

Arquivo onde se pode inserir textos, imagens, vídeos, entre outros. Possuem hipertextos que permitem a navegação entre páginas ou seções.

Websites ou Sites

Conjunto de páginas armazenadas em uma pasta ou subpasta relacionada de um servidor.

Arquitetura Cliente-Servidor e seus profissionais

Client-Side
Tecnologias: HTML, CSS, JavaScript.
Profissional: Front-End.
Server-Side
Tecnologias: Java, MySQL, Python.
Profissional: Back-End.
FullStack
Domínio de ambas as tecnologias, desde o lado do cliente até o lado do servidor.

HTML

Linguagem de Marcação de Hipertexto, baseada em tags e desenvolvida por Tim Berners-Lee

O HTML é usado na publicação de hipertexto, vinculados através de hyperlinks.


Tags

Realiza a marcação de conteúdo, possuem funcionalidades específicas, sem o seu uso é exibido um texto plano.

Código:
<tag> conteudo </tag>

Podem possuir atributos, que fornecem informações extras.

Código:
<tag atributo="valor"> conteudo </tag>

Hyperlinks

É usado como referência dentro de um documento para outras partes do mesmo, para outros documentos ou um endereço qualquer.
São divididos em:

  • Remotos
  • Internos
  • Locais
  • Com endereço ou arquivos

Marcação de Hyperlinks

A marcação hyperlinks é feita utilizando a tag <a> e a propriedade href que realiza a referência do conteudo

Sintaxe

<a href="endereço">Conteúdo Clicavel</a>

Hyperlinks Externos ou Remotos

São links que fazem referência a sites na web, no href usamos uma URL - Uniform Resource Locator para referenciar.

Sintaxe

<a href="URL">Conteúdo Clicavel</a>

Uso do target=_blank: adicionar no seu hyperlink, quando clicado o navegador abri-rá uma nova guia.

<a href="URL" target="_blank>Conteúdo Clicavel</a>

Uso do target="_self":Ao adicionar no seu hyperlink, quando clicado o navegador abri-rá uma nova guia.

<a href="URL" target="_self>Conteúdo Clicavel</a>

Uso do rel="external":Define o hyperlink como externo.

<a href="URL" target="_blank rel="external">Conteúdo Clicavel</a>

Uso do hreflang="Linguagem": Ao adicionar no seu hyperlink, quando clicado o navegador abri-rá uma nova guia.

<a href="URL" target="_blank rel="external" hreflang="Linguagem">Conteúdo Clicavel</a>

Hyperlinks Internos

São links que apontam para lugares diferentes em um mesmo documentos HTML.

Sintaxe

<a href="#referencia">Conteúdo Referenciado</a>

<a name="referencia">...</a>

Hyperlinks Locais

São links que apontam para paginas diferentes em HTML de um mesmo site.

Sintaxe

<a href="arquivo.html">Conteúdo Referenciado</a>

Hyperlinks em Imagens

São utilizados para criar hyperlink em uma imagem, utilizam a tag de imagem e a tag de hiperlink.

Sintaxe

<a href="referencia"><img src="imagem.jpg"></a>

Hyperlinks para Download de Arquivos

Apontam para arquivos e realizam o download dos arquivos selecionados.

Sintaxe

<a href="arquivo.extensão" download="arquivo.extensão" type="Media-Types"><Conteudo</a>

Media-Types - é um identificador padrão usado na Internet para indicar o tipo de dado que um arquivo contém. Media-Types

Hyperlinks para Endereço Eletrônico

Apontam para um endereço eletrônico dos arquivos selecionados.

Sintaxe

<a href="mailto:email@email.com"><email@email.com</a>
