Formulário

Definição

Formulario é uma estrutura do HTML. Que permite a entrada de dados pelo usuário na página HTML.

Estrutura

Exemplo

DOM - Document Object Model

API que tem por finalidade auxiliar no desenvolvimentzo de aplicações.

Simplifica a forma de acessar a estrutura de um documento HTML. Facilitando criar e modificar.

Coloca o documento do HTML como um objeto e cada tag como um nó.

Sintaxe

document.getElementById("id")
document.getElementsByName("tag)
innerHTML = "O que será exibido"
