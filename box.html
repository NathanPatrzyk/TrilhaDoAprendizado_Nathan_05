Box Model

O modelo das caixas em CSS, descreve os boxes geradas pelos elementos HTML detalhando as opções de ajuste de margens, bordas, padding e conteúdo para cada elemento.

Entender o Box Model é fundamental para construção de layout em sites.

Posicionamento

Estático - permanece na posição em que foi definido
Relativo - posicionado em relação onde foi definido
Absoluto - pode ser posicionado em qualquer lugar do documento
Fixed - permanece fixo durante a rolagem do conteúdo
Sticky - permanece fixo durante a rolagem a partir de onde foi definido

Float

O float é utilizado para determinar a forma como um elemento deve flutuar em determinado contêiner.

Usado para envolver o texto em imagens

Pode conter os valores:

  • none: Forma "padrão" de flutuação, não flutua;
  • inherit: Herda o valor flutuante de seu pai;
  • left: Flutua a esquerda do seu contêiner;
  • right: Flutua a direita do seu contêiner;
